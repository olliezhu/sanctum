# Privacy Policy

Sanctum does not collect, store, transmit, or process any personal data.

## Data access

The extension runs only on:
- `reddit.com`
- `twitter.com`
- `x.com`

It injects a small amount of CSS to hide feed UI and other distracting UI elements on these domains.  
It does not read or analyze page contents, user identifiers, cookies, or browsing history.

## Data collection

Sanctum collects no user data.  
It sends no data to external servers.  
It does not use analytics or tracking of any kind.
It does not access cookies, browsing history, localStorage, accounts, or authentication tokens.
