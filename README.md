# sanctum

Blocks the feed and some noisy elements on social media websites. Currently only supports Reddit and Twitter/X.

You can install this extension from the Chrome Web Store [here](https://chromewebstore.google.com/detail/sanctum/aipdmodejpngkbhggilommlcjbbclkmp
), or by downloading this repository and selecting it when you `Load unpacked` in your chromium browser at `chrome://extensions/`.

Forked from [twitter-x-feed-blocker](https://github.com/akhileshthite/twitter-x-feed-blocker).
